# android_tools

#### 1. Android APK 批量重签名工具

##### 操作方式：
	（1）安装 apktool；
	（2）下载 debug.keystore 和 resignature_apk.sh，将其放在需要重新签名的 apk 相同目录下;
	（3）执行 “$ bash resignature_apk.sh” 即可；


##### 参考资料链接：http://www.cnblogs.com/findyou/p/3801273.html
